# Changelog

## 1.0.0 - 2026-05-29

- Initial commercial MVP for Substack Publication and Post Scraper.
- Added public publication URL and direct post URL inputs.
- Added public RSS feed discovery and parsing.
- Added public post page extraction for title, URL, slug, author, dates, excerpt, image, tags, visible counts, and public article text.
- Added `accessStatus`, `isPaidPreview`, and `isPubliclyReadable` fields to clearly mark public, preview-only, and unavailable posts.
- Added Apify input schema, dataset schema, output schema, examples, Store listing copy, README, launch checklist, and tests.
